# X-tools 1.8.0 发布说明（2026-04-16）

## 概述
X-tools 1.8.0 版本在 1.7.0 的基础上，带来了文件拖拽导入/导出功能的全面完善、文件夹统计功能的增强、搜索体验的进一步优化，以及多项稳定性和性能改进。本版本专注于提升本地资料库管理的便捷性和效率，为用户提供更加流畅和强大的文件操作体验。

## 主要更新

### 1. 文件拖拽导入/导出功能（全新）
- **文件拖拽导入** - 支持从外部系统拖拽文件和文件夹到 X-tools 应用中，实现快速导入
- **文件拖拽导出** - 支持从 X-tools 中拖拽文件到外部系统，实现真正的操作系统级文件拖拽

### 2. 文件夹统计功能增强
- **文件数量统计优化** - 改进文件夹中文件和子文件夹的计数准确性
- **字数统计统一** - 统一文本文件的字数统计逻辑，支持中文字符和英文单词分别统计
- **批量字数统计** - 支持选中多个文件进行批量字数统计
- **扩展名分类统计** - 按文件扩展名分类展示统计结果，便于分析文件类型分布

### 3. 搜索功能改进
- **搜索预览体验优化** - 改善搜索结果的预览显示效果
- **视频文件直接预览** - 支持在搜索结果中直接预览视频文件内容
- **字幕文件搜索增强** - 支持 LRC 字幕文件的搜索和预览

### 4. 视频播放功能优化
- **LRC 字幕支持** - 新增对 LRC 歌词字幕文件的支持
- **字幕显示状态保持** - 修复视频播放时字幕显示状态丢失的问题
- **自动播放问题修复** - 解决视频自动播放的相关问题
- **字幕搜索功能完善** - 增强字幕搜索的准确性和响应速度

### 5. 文件管理功能增强
- **批量操作支持** - 在文件夹视图中支持文件的批量删除、移动和打开操作
- **标题栏拖动修复** - 修复标题栏拖动失效的问题
- **文件树右键菜单优化** - 改善右键菜单的分隔符显示和菜单项布局

### 6. 新功能引入
- **打开终端功能** - 添加打开终端功能，方便在当前位置执行本地命令
