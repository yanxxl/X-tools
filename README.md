# X-tools

一个 基于 Electron 的本地资料库浏览工具。

## 主要功能

### 已实现

- 📚 **资料库浏览** - 高效浏览本地文档、笔记和学习资料
- 📖 **多格式预览** - 支持 Markdown、PDF、图片、视频等学习资料预览
- 📚 **搜索** - 全文搜索，文件名搜索，快速查找定位
- 📚 **视频字幕** - 字幕展示，快速切换，搜索

### TODO

- 📚 **本地词典** - 从本地 md 文件生成词典，快速查词
- 📚 **视频** - 导出视频片段
- 📚 **格式** - 支持更多格式，docx，等
- 📚 **搜索** - 支持更多格式，pdf，docx，等

## 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Electron + Node.js
- **构建**: Electron Forge

## 快速开始

```bash
npm install
npm start
```

## 适用场景

- 学习资料快速浏览
