# X-tools

一个基于 Electron 的本地资料库浏览工具，专为学习和知识管理设计。

## 主要功能

- 📚 **资料库浏览** - 高效浏览本地文档、笔记和学习资料
- 🗂️ **懒加载目录树** - 快速导航大型资料库，支持按需加载
- 📖 **多格式预览** - 支持PDF、图片、视频等学习资料预览
- 🖥️ **自适应界面** - 根据显示器分辨率自动调整窗口大小

## 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Electron + Node.js
- **构建**: Electron Forge

## 快速开始

```bash
npm install
npm start
```

## 适用场景

- 个人知识库管理
- 学习资料整理与浏览
- 文档库快速检索
- 本地资料预览工具
