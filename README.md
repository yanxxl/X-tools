# X-tools

一个本地资料库浏览工具。

一般每周发布一个版本，添加新功能，修复 bug，优化性能，请保持关注，及时升级。

## 主要功能

### 已实现

- 📚 **资料库浏览** - 一颗巨大的文件树，高效浏览本地文档、笔记和学习资料
- 📖 **多格式预览** - 支持 Markdown、PDF、图片、视频，office（docx，xlsx，pptx）等学习资料预览
- 📚 **搜索** - 全文搜索，文件名搜索，快速查找定位，支持多格式搜索
- 📚 **词典** - 支持，快速查词，选中文本自动查词
- 📚 **视频** - 字幕展示，快速切换，搜索过滤
- 📚 **图片** - 缩放，旋转等

## 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Electron + Node.js
- **构建**: Electron Forge

## 快速开始

```bash
npm install
npm start
```

## 适用场景

- 学习资料快速浏览
