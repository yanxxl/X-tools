# X-tools

一个本地资料库浏览和管理工具。

一般每周发布一个版本，添加新功能，修复 bug，优化性能，请保持关注，及时升级。

## 主要功能

### 已实现

- 📚 **文件管理** - 支持完整的增删改查操作，支持文件/文件夹创建、重命名、删除、拖拽，过滤查询等
- 📖 **资料库浏览** - 支持文件树展示，高效浏览本地文档、笔记和学习资料
- 📖 **多格式预览** - 支持 Markdown、PDF、Office（docx，xlsx，pptx）、文本、视频、图片等
- 🔍 **全文搜索** - 支持多格式，全文搜索，支持多线程，速度快
- 📖 **Markdown** - 支持大纲，编辑，朗读等
- 📺 **视频播放** - 支持多字幕展示，循环播放，快速切换，搜索过滤等
- 📚 **词典查词** - 支持 Markdown 文件为词典，快速查词，选中文本自动查词
- 🖼️ **图片查看** - 支持缩放，旋转等操作

## 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Electron + Node.js
- **构建**: Electron Forge

## 快速开始

```bash
npm install
npm start
```
